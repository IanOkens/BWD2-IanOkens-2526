Aangepaste kleuren met variabelen

Voorbeeld op getbootstrap.com/docs/5.3/components/alerts/

Dit voorbeeld demonstreert voornamelijk het gebruik van variabelen in css.

Wanneer je gebruik maakt van een framework als Bootstrap is het meestal beter om met het framework te werken en niet tegen het framework door je eigen css te schrijven. Maar je kan.

Een betere manier is door dit via de buildtools te doen en rechtstreeks de CSS variabelen van Bootstrap aan te passen, voor het build-proces.
